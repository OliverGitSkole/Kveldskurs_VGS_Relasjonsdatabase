Send inn spørsmål

Du kan sende inn spørsmål via skjemaet til høyre.
Navn er obligatorisk – hvis du ikke ønsker å oppgi navnet ditt, kan du skrive "anonym" i feltet.

Ofte stilte spørsmål

  • Hvordan kan jeg melde barnet mitt på kurs? Du kan ringe til +47 909 78 129, for å melde på kurs
  • Hvordan sender jeg inn et spørsmål? Fyll ut skjemaet på siden med navn, tema og spørsmålet ditt, og trykk "Send inn".
  • Kan jeg være anonym? Ja! Skriv "anonym" i navnefeltet hvis du ikke ønsker å oppgi navnet ditt.
  • Hvor lenge tar det å få svar? Vi prøver å svare innen 1–2 virkedager.
